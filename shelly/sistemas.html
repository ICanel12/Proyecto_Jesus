¿Qué es un sistema de ecuaciones?

Un sistema de ecuaciones es un conjunto de dos o más ecuaciones con multiples incógnitas sobre las cuales se quiere encontrar un solución común.

Una solución es todo par de números que cumple la ecuación.

 Existen diferentes métodos de resolución:

  • •  Sustitución.
  • •  Reducción.
  • •  Igualación.



1. Sustitución


El metodo de sutitución consiste en despejar una de las incognitas en una de las ecuaciones y sustituirlo en la otra ecuacion.

Ejemplo

Tenemos el siguiente sistema de ecuaciones:

x - y = 2

x + y = 0


Despejamos la primera ecuacion:

x = 2 + y

Luego sustituimos x en la segunda ecuacion:

(2 + y) + y = 0

Operamos

2 + 2y = 0

Resolvemos para "y"

2y = -2

y = -2/2

y = -1

Ahora que tenemos "y", podemos resolver para "x", sustituimos el valor de "y" en el primer depeje que hicimos para "x".

x = 2 + (-1)

Operamos

x = 1

Entonces llegamos al resultado de que

x = 1

y = -1




2. Reducción


Consiste en sumar o restar nuestras ecuaciones de tal manera que una de las incognitas desaparezca

Ejemplo

Tenemos el siguiente sistema de ecuaciones:

x - y = 2

x + y = 0


Sumamos las dos ecuaciones:

(x + y) + (x - y) = 0 + 2

Operamos

2x = 2

Ahora que ya solo hay una incognita resolvemos para la misma

x = 2/2

x = 1

Para terminar sustituimos el valor que hemos calculado en la ecuacion despejada para la otra incognita.

y = -x

y = -(1)

y = -1

Entonces llegamos al resultado de que

x = 1

y = -1




3. Igualación


Este metodo consiste en despejar la misma incognita en ambas ecuaciones y posteriormente igualar ambos despejes.

Ejemplo

Tenemos el siguiente sistema de ecuaciones:

x - y = 2

x + y = 0


Despejamos la primera ecuacion para x

x = 2 + y

Despejamos la segunda ecuacion para x

x = -y

Igualamos las dos ecuaciones

2 + y = -y

Resolvemos para y

y + y = -2

2y = -2

y = -1

Ahora que tenemos y, lo sustituimos en cualquiera de los despejes que hicimos para x

x = -(-1)

x = 1

Entonces llegamos al resultado de que

x = 1

y = -1
